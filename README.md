SAP BTP Leave Approval Workflow

Overview
This project demonstrates a Leave Approval Workflow implemented using SAP Build Process Automation.

Features
- Employee leave request form
- Manager approval process
- Workflow design using low-code tools
- Deployment on SAP BTP Trial

Technologies Used
- SAP BTP
- SAP Build Process Automation
- Forms
- Workflow

Process Flow

Start
↓
Leave Request Form
↓
Manager Approval
↓
Approve / Reject
↓
End

Learning Outcomes
- Process modeling
- Form design
- Role management
- Environment deployment
- Workflow lifecycle management
